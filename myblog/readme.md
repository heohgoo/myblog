장고를 이용하여 자신만의 게임 블로그를 개발하였습니다.
간단하게 프론트쪽은 bootstrap쪽에서 블로그 예시 html,css파일을 가져와 사용하였고, 백쪽은 장고로 api서버를 구축하여 구현하였습니다.
admin을 이용한 로그인 기능과 post를 간단히 미리보기로 확인하는 기능, 로그인 후 post를 작성할 수 있는 기능까지 추가되었습니다.
